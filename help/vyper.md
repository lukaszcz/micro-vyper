# Vyper Syntax

This plugin registers syntax highlighting for Vyper source files.

Files ending in `.vy` and `.vyi` are detected as `vyper`.

The syntax definition uses Micro highlight groups instead of hardcoded colors,
so the active colorscheme controls the final appearance.

## Install

Clone or install this plugin into Micro's plugin directory:

```
~/.config/micro/plug/vyper/
```

Restart Micro after installation.

For manual syntax-only installation, place `syntax/vyper.yaml` in:

```
~/.config/micro/syntax/vyper.yaml
```
